HOW TO RUN:
===========

1. START XAMPP
   - Open XAMPP Control Panel
   - Start Apache
   - Start MySQL

2. START BACKEND (Terminal 1)
   cd C:\xampp\htdocs\EventManagementSystemOJT\backend
   C:\xampp\php\php.exe artisan serve

3. START FRONTEND (Terminal 2)
   cd C:\xampp\htdocs\EventManagementSystemOJT\frontend
   npm run dev

4. OPEN BROWSER
   http://localhost:5173

TROUBLESHOOTING:
================
Problem: "Could not open input file: artisan"
Solution: Use full path: C:\xampp\php\php.exe artisan serve

Problem: "404 Not Found"
Solution: Make sure backend is running on port 8000

Problem: "500 Error"
Solution: Check backend terminal for errors

Problem: "Can't register"
Solution: Email must be: main.firstname.lastname@cvsu.edu.ph

Problem: "Database error"
Solution: 
  1. Make sure MySQL is running
  2. Database "event_management" exists
  3. Run: C:\xampp\php\php.exe artisan migrate
