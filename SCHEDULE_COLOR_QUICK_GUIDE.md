# Class Schedule Color Indicators - Quick Guide

## What's New? 🎨

Your class schedule now features automatic color indicators! Each class is assigned a unique color to help you visually distinguish between different classes at a glance.

## How to See It

1. Go to **Account Dashboard** (click your profile icon → Account)
2. Scroll to the **Class Schedule** section
3. Look for the colored circles next to each class time

## How It Works

### Deterministic Color Assignment
- Colors are assigned based on your **class description**
- The same class gets the same color on all days
- Example: If "Data Structures" is on Monday and Wednesday, it will have the same color both days
- Colors are case-insensitive (e.g., "data structures" = "Data Structures")

### Automatic Assignment
- When you save your schedule, each unique class automatically gets a color
- First unique class → First color (Green)
- Second unique class → Second color (Blue)
- And so on through 10 distinct colors
- No manual selection needed - it's completely automatic!

### Color Persistence
- Your colors stay the same every time you log in
- Same class = Same color across all days
- Colors don't change unless you modify class descriptions

## FAQs

**Q: Will the same class have the same color on different days?**  
A: Yes! That's the key feature. If "Data Structures" appears on Monday and Wednesday, it will have the same color both days.

**Q: Can I choose my own colors?**  
A: Currently, colors are automatically assigned based on class names. Custom color selection may be added in a future update.

**Q: What if I have more than 10 unique classes?**  
A: The system will cycle through the color palette, so some classes may share colors.

**Q: Will my colors change if I edit my schedule?**  
A: Colors are based on class descriptions. If you keep the same class names, they'll keep the same colors. If you rename a class, it will get a new color.

**Q: Are colors case-sensitive?**  
A: No! "Data Structures", "data structures", and "DATA STRUCTURES" will all get the same color.

**Q: Can I see these colors in the calendar view?**  
A: Currently, colors are visible in the Account Dashboard. Calendar integration may be added in future updates.

## Need Help?

If you don't see color indicators:
1. Make sure you've saved your schedule at least once
2. Try refreshing the page (Ctrl+R or Cmd+R)
3. Clear your browser cache if colors don't appear
4. Contact your administrator if issues persist
