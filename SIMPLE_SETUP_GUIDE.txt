========================================
SIMPLE LARAVEL + REACT + TAILWIND SETUP
========================================

This is a clean, minimal setup guide for your event management system.

WHAT YOU NEED:
--------------
1. XAMPP (Apache + MySQL)
2. Composer installed
3. Node.js installed


STEP-BY-STEP SETUP:
===================

STEP 1: CREATE DATABASE
-----------------------
1. Start XAMPP (Apache + MySQL)
2. Open: http://localhost/phpmyadmin
3. Click "New"
4. Database name: event_management
5. Click "Create"


STEP 2: SETUP BACKEND (Laravel)
--------------------------------
1. Open Command Prompt
2. Navigate to project:
   cd C:\xampp\htdocs\EventManagementSystemOJT

3. Remove old backend (if exists):
   rmdir /s /q backend

4. Create fresh Laravel:
   composer create-project laravel/laravel backend

5. Navigate to backend:
   cd backend

6. Install Sanctum:
   composer require laravel/sanctum

7. Configure database:
   - Open backend/.env in text editor
   - Find these lines and make sure they match:
     DB_CONNECTION=mysql
     DB_HOST=127.0.0.1
     DB_PORT=3306
     DB_DATABASE=event_management
     DB_USERNAME=root
     DB_PASSWORD=

8. Run migrations:
   php artisan migrate

9. Publish Sanctum:
   php artisan vendor:publish --provider="Laravel\Sanctum\SanctumServiceProvider"
   php artisan migrate


STEP 3: SETUP FRONTEND (React)
-------------------------------
Frontend is already set up! Just install dependencies:

1. Open NEW Command Prompt
2. Navigate to frontend:
   cd C:\xampp\htdocs\EventManagementSystemOJT\frontend

3. Install dependencies (if not done):
   npm install


STEP 4: RUN THE APPLICATION
----------------------------
You need 2 terminals open:

Terminal 1 - Backend:
  cd C:\xampp\htdocs\EventManagementSystemOJT\backend
  php artisan serve

Terminal 2 - Frontend:
  cd C:\xampp\htdocs\EventManagementSystemOJT\frontend
  npm run dev

Then open browser: http://localhost:5173
