Event Files Test

To test the event file functionality:

  1. Create an event with files using the EventForm
  2. View the event in the calendar
  3. Click on the event to open the detail modal
  4. Verify files are displayed with download links

Expected Features:

  • File upload in EventForm (drag & drop or click)
  • File preview in form
  • File display in event detail modal
  • Download links for files
  • File type icons (PDF vs Image)
